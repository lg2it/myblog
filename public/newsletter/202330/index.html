一、我如何用 Heptabase Journal 辅助写作 #

首先，在阅读过程中，积极寻找并记录灵感，包括感兴趣的内容、引起我注意的点或需要深入探索的部分。

我基本都是把需要阅读的内容加入到 Readwise Reader 中，这样就方便我在阅读的过程中直接标注，然后它们会迅速同步到 Heptabase 的 Highlight 中。

这些读书笔记，有一些可能是就事论事，我读到了什么，有什么感受。还有一些则可能会发散。无论什么样的笔记，总之记录是最关键的。

其次，灵感不是凭空而来的，需要积累。把 Journal 当作稿纸，它是一个跳板，一个开始思考和写作的提示。

在 Journal 中除了读书笔记，还要记录简短的日常观察、事实或思考。日常的思考和记录会催生许多独到的观点和想法。

在 Heptabase 中可以很轻松的将一些内容选中后变成卡片，所以 Journal 中的这些记录，有一些会重新排列组合后形成一张新的卡片，有的会更新到已有的卡片，而有一些则会留在原地。

不过，留在 Journal 中的内容也有区别，除了不会被我轻易删除外，还会给它们更多的形式。比如：

  • 基本成熟但还不足以形成卡片的内容会被标绿 🟢
  • 有待进一步发展的内容会被标黄 🟡
  • 遇到了障碍的内容会被标红 🔴
  • 还有一些内容则会标记上删除线。

最后，开始正式写作流程前，我会新建一个白板，然后将需要用到的卡片都拖入白板中。

因为卡片基本都是通过 Journal 新建的，我可以在 Info 中看到这些卡片都来自哪一天的 Journal。我也会把这些 Journal 拖入到白板中，也许里面有一些零散的内容我会用到。这也就是为什么我一直在说不轻易删除 Journal 中的内容的原因，哪怕那些在当时看起来是无用的。

二、我犯了一个严重错误 #

在十一假期的后半段，我开始了一个新的项目的筹备。这个月下旬要拍摄一个短片，展示现在数码产品上的无障碍功能如何为视障人群带来便利。但是，直到上周五的中午，我才突然意识到自己犯了一个巨大的错误——我并没有真正站在视障人群的角度去思考。

事实上，我没有真正尝试过数码产品上的无障碍功能。我甚至没有尝试戴上眼罩，让自己学习和体验盲人的生活。我所做的，只是站在一个健全人的角度，以我自己的视角去考量一切，然后与拍摄的对象沟通，仿佛我已经知道了所有的答案。我所做的，只是自我感动。找了一些样片，响拍摄对象了解了一些基本的信息……但其实我并没有深入了解他们的真实需求和感受。

这个错误，是非常严重的。我以前总是会亲自去体验，然后带着这些经验去与拍摄对象沟通。但这一次，我为什么没有做到？是因为我的自我意识太强烈了，甚至到了一种歧视和冷漠的地步。

而由这个错误所引发的是一系列的 trouble。比如，我会以一个「健全人」的视角审视视障人群，一定要套上一个自立自强的故事，去展示他们的身残志坚。凭什么残障人群就得身残志坚呢？这种视角从本质上来说，还是把他们是为异类。再比如，我并没有真的关注他们到底是不是觉得无障碍功能好用，而只是以我自己的臆想去要求拍摄对象配合。那这样拍出来的短片，且不说没有人文关怀，可能就连基本的实事都是虚假的。

Oringe 在 Twitter 上发过一条推文，他说：

「换位思考」是个很理想的建议，但几乎不可能的事。每个人对世界的认知都是不同的，你怎么可能真正地站在别人的角度去看待这个世界呢？只有当对方愿意为你搭建一座桥，让你进入他的世界，同时你也愿意走进去，这样的换位思考才有可能实现。

确实，要真正地换位思考是很困难的，我承认自己的局限性，慈悲心也不够。但至少，我可以尝试理解和体验，带着这样的心态去与短片中关注的对象沟通和合作。我需要真正地倾听他们的声音，而不是只是满足于自己的想法和观点。

三、谈唐诗 #

诗词是生活的调味料，它能为生活带来情感的丰富和思想的深度，正如调味料为食物增添味道。上周和朋友交流时，聊了几个小时的唐诗。我简单整理了一下，形成文字。

唐诗一直以李杜为尊，李白为诗仙，杜甫为诗圣。

人们深爱李白的原因很有可能是因为他太不「中国」了。有谁能做到「天子呼来不上船，自称臣是酒中仙」（这是杜甫描述的李白）？就算是竹林七贤、陶渊明也做不到。

这倒是非常有先秦时的感觉。李白如果能遇到庄子，两人一定是「臭味相投」的。两人如果能遇见再早一些的许由，那「三人行，必有我师焉」恐怕就不是儒生说的了。

而杜甫则太「中国」，所以只能作圣。太「中国」就意味着被儒家礼教绑架。杜甫一辈子都没有逃出儒家礼教的牢笼。

安史之乱时如果肯往前走一步，或许也能成仙。但终究没能做到。正因为此，他用毕生心血写就了全唐音韵最圆通的诗，来唏嘘那个「懦弱」的自己。

风急天高猿啸哀，渚清沙白鸟飞回。 无边落木萧萧下，不尽长江滚滚来。 万里悲秋常作客，百年多病独登台。 艰难苦恨繁霜鬓，潦倒新停浊酒杯。

唐诗的风华，除了李杜，初唐时有王勃独占四杰鳌头（初唐四杰并非指其诗的优秀，而是指骈文和赋）。奈何王勃早殇，魂断南海，给了张若虚「孤篇压全唐」的机会。

木心说「陈子昂，是唱唐代文学宣叙调的男高音、领唱者。」作为「唐诗诗祖」的陈子昂，风骨傲世，对扫除齐梁浮艳纤弱的形式主义诗风具有拓疆开路之功。

盛唐时，王维与岑参，一个跑去田园牧歌，自成一派；一个在边塞慷慨激昂。算是从李杜那里抢占了一点地盘。

中唐的诗人中，白居易最为后世偏爱，或许与他写的真好有关。其他的那些就像大唐的气数一样，上气不接下气。

晚唐则只剩下杜牧和李商隐了，却唯有李商隐直通现在，与今人神魂相交。

四、夜航船 #

阅读 #

  • 与社交媒体的相关的研究一直是我关注的重点，上周有三篇文章引起我的注意。
    • 第一篇 研究了社交媒体算法如何扭曲人们从彼此中学习的方式。文章指出，人们与在线算法的日常交互影响了人们从他人那里学习的方式。这对社会认知产生了负面的影响，包括社会误解和类似的其他问题。在社交媒体平台上，算法主要是为了扩大那些能够维持用户参与度的信息，也就是说它们让人们继续点击和消耗内容。尽管这种方式可能会带给人们某种娱乐和参与感，但却可能歪曲人们的学习和认知，因为这种算法可能只强调或展示某一面的信息，而忽视了其他重要的观点。
    • 第二篇 的标题就很吸引眼球，If you want to be a creator, delete all social media platforms。文章认为，社交媒体平台的创作者会推荐在多个平台上发布内容，但这实际上会浪费你的时间。建议只保留你最擅长和喜欢的两个平台，这样可以更好地把精力集中在这两个平台上。
    • 我们所熟悉的社交网络，也就是那个在其中消费帖子并与他人互动的地方，似乎已经结束。为什么社交网络不再像以前那样有趣了？ 像 Twitter 这种平台的衰落预示着互联网的新时代，这个时代比过去的乐趣要少，少数几个巨大的社交网络使我们的体验同质化。尽管今天的互联网充满了比以往更多的内容，但感觉上却更加空虚，不再像以前那样随意地提供信息。推荐流量的减少破坏了媒体的商业模式，进一步降低了在线原创内容的质量。
  • 「Heptabase 是更好的笔记软件吗？其实我们都问错了问题」是一篇探讨笔记软件选择的文章。文章指出我们常见的问题是「我应该换软件吗？」或「这个软件有什么功能？」，但这些问题过于局限。文章提到，真正重要的是「对复杂的事物建立深度理解的能力」，而笔记软件只是工具之一。文章解释了「对复杂事物建立深度理解的能力」的重要性，并指出这种能力并非天赋，而是需要运用工具和进行思考和整理。最后，文章提醒我们不要把笔记软件视为终点，而是要思考如何更好地理解资讯。
  • Coding was hard until I learned these 2 things 主要介绍了作者从一开始学习编程很困难，到后来成功找到工作需要的两个要点: 1. 培养增长心态。这意味着看待自己的编程能力可以通过努力来提高，而不是天生就有或没有。当遇到困难时应该看作学习机会，而不是自我否定。 2. 将编程变成日常习惯。通过设置明确的时间和地点，例如每天下午 4 点在咖啡馆学习编程，来确保每天练习。将学习编程与自己喜欢的事情结合，例如只有学习编程时才允许自己喝咖啡，可以帮助培养这个习惯。
  • 思考工具是一个新兴领域，旨在开发和改进能够增强人类思考和生产力的工具。据 Andy Matuschak 介绍，这个领域的特点是一系列的活动，包括识别强大的洞察力，构建表达这些洞察力的系统，观察它们在真实环境中的使用，以及从这些观察中提炼出可推广的洞察力。然而，许多对思考工具感兴趣的个人和科技公司并未有效地执行所有这些步骤，导致该领域的进展缓慢。作者认为，思考工具的研究应该努力生成启发式的陈述，连接理论主张、设计方法和期望的结果。文章强调了几个体现了思考工具工作全周期的当代项目。
  • 猫咪是完美的，您认同这个观点吗？进化生物学家 Anjali Goswami 认为，从家猫到老虎，猫科动物在进化意义上是完美的。她解释说，尽管毛色有所不同，但猫的基本特征，如圆头，贯穿其一生都是相同的。这是因为猫是高度专业化的捕食者，只保留了用于切割肉类的必要牙齿。猫通过在特定生态位上表现出色来实现进化成功，而其他试图模仿猫的哺乳动物群体则无法竞争。Goswami 还指出，与其他哺乳动物相比，猫的进化速度较慢。
  • Career Advice 一文为寻求就业指导的年轻人提供职业建议。作者警告说，工作可能会改变一个人，并建议观察年长的同事以获得对可能的未来路径的洞察。作者还建议走出支持结构，挑战自我，做出的选择不仅仅是为了赚钱。作者鼓励在工作之外追求那些不舒服但有意义的经历。
